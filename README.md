## Token Gen: 

### Features:

- `Meilleur générateur de tokens`
- `Tokens pas vérouillés`
- `Choix du pseudo`
- `Différents sites d'emails`
- `Les tokens sont dans un fichier`
- `Tokens envoyer à un webhook`
- `Hcaptcha bypass`
- `Verificateur d'email automatique`
- `et bien plus`

### Requirements:
- Windows 10 / 11
- NodeJS
- npm i
